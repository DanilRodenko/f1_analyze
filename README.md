# Formula 1 Data Analysis Project

## 📌 Overview
This project analyzes historical Formula 1 race data with a focus on:
- Race chaos and unpredictability
- Driver and constructor performance (Top-3 finishes)
- Reliability and DNFs
- Era-based comparisons
- Impact of technical regulations and engines
- Weather influence (optional module)

The goal is to combine exploratory data analysis with structured metrics that explain
*why some races, eras, and tracks are more chaotic or stable than others.*

## 🧠 Key Analyses
- **Race Chaos Index** (DNF rate + order disruption)
- Comparison of chaos across F1 eras
- Track-level stability vs unpredictability
- Reliability analysis by engine manufacturer
- Driver & constructor Top-3 consistency
- Weather-based classification (Dry / Wet / Variable)

## 🗂 Project Structure
- `notebooks/` — exploratory analysis and visual storytelling
- `src/` — reusable data processing and analysis logic
- `datasets/` — raw and processed data (raw ignored in git)

## 🛠 Tech Stack
- Python
- pandas, numpy
- matplotlib, seaborn
- scikit-learn (for auxiliary analysis)
- sentence-transformers (weather classification)
